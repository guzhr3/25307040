# 记账助手工程说明

## 项目简介

记账助手是一款使用 HarmonyOS ArkTS Stage 模型开发的本地记账 App。应用数据通过鸿蒙本地关系型数据库 RDB 独立存储，不依赖服务端，适合记录日常收入、支出和查看账单统计。

项目已实现当月账单查看、添加记账、收入支出分类、账单详情、数据统计等基础功能，并使用断点系统支持手机和平板等不同屏幕尺寸。

## 功能概览

- 当月收入、支出总览
- 按日期分组展示账单记录
- 添加支出账单和收入账单
- 支持账单分类、金额、日期、备注
- 支持删除账单记录
- 支持按月、按年统计账单数据
- 支持按日期和按分类查看统计结果
- 使用本地 RDB 数据库保存账单数据
- 使用 ArkTS Stage 模型组织应用生命周期和页面

## 核心目录说明

### AppScope

`AppScope` 保存应用级配置和全局资源，主要包括应用名称、图标、包信息等内容。

### entry

`entry` 是主模块，包含应用主要页面、组件、数据模型、业务服务和资源文件。

### entry/src/main/ets/entryability

`EntryAbility.ets` 是应用入口能力，负责应用生命周期处理、窗口创建、本地数据库初始化、偏好设置初始化等工作。

### entry/src/main/ets/pages

页面目录，包含主要路由页面：

- `Index.ets`：应用主页面，承载首页和统计页 Tab。
- `AddAccount.ets`：添加记账页面。
- `AccountDetail.ets`：账单详情页面。

### entry/src/main/ets/views

视图目录，保存主页面中的功能视图：

- `home.ets`：首页账单列表和本月收支概览。
- `stats.ets`：统计页，提供月度和年度统计。

### entry/src/main/ets/components

组件目录，保存页面中复用的 UI 组件：

- `AccountItem.ets`：账单条目组件。
- `Navigator.ets`：返回导航组件。
- `DatePickerDialog.ets`：日期选择弹窗。
- `PickerDialog.ets`：通用选择弹窗。
- `ToastDialog.ets`：提示弹窗。
- `addAccount/AccountForm.ets`：添加账单表单。
- `stats/StatsHeader.ets`：统计页顶部筛选组件。
- `stats/StatsDetailHeader.ets`：统计详情筛选组件。

### entry/src/main/ets/models

数据模型目录，定义账单、分类、统计等数据结构。

- `models/account/account.ets`：账单相关枚举和接口。
- `models/account/accountPO.ets`：账单数据库持久化对象。
- `models/account/accountModel.ets`：账单数据库操作模型。
- `models/accountType/accountType.ets`：收入和支出分类数据。
- `models/stats/stats.ets`：统计类型和分类类型。

### entry/src/main/ets/service

业务服务目录。`service/account.ets` 封装账单新增、删除、按月查询、按年查询、统计计算等业务逻辑。

### entry/src/main/ets/utils

工具目录，包含数据库、日志、偏好设置和断点系统等工具。

- `DBUtil.ets`：RDB 数据库初始化和通用查询封装。
- `PreferenceUtil.ets`：首选项工具。
- `BreakpointSystem.ets`：屏幕断点监听。
- `Logger.ets`：日志工具。

## 页面说明

### 首页

首页用于展示当前月份的账单数据，包括本月支出、本月收入和按日期分组的账单列表。每条账单展示分类图标、分类名称、备注、金额等信息。

首页只展示当前月份的记录。如果添加账单时选择了其他月份，该记录不会出现在首页本月列表中，但可以在对应时间范围的统计或详情中查看。

### 添加记账

添加记账页支持录入收入和支出。用户可以选择账单分类，输入金额，选择日期，并填写备注。保存后数据写入本地 RDB 数据库。

支出分类包括餐食、医疗、交通、住房、学习、购物、娱乐等。收入分类包括工资、奖金、利息、投资、兼职、红包转账等。

### 统计页

统计页支持按月和按年查看账单统计。统计结果可以按日期或按分类展示，方便查看不同时间段和不同分类的收入支出情况。

### 账单详情

账单详情页用于展示某一天或某个月的账单明细，配合统计页使用，方便从统计结果进入具体记录列表。

## 数据存储

项目使用 HarmonyOS 关系型数据库 RDB 保存账单记录。账单数据主要包含以下字段：

```text
id           账单 ID
type         分类类型
accountType 账单类型，支出或收入
amount       金额
time         账单日期
description 备注
createTime   创建时间
```

数据库初始化在应用启动阶段完成，业务层通过 `AccountModel` 和 `AccountService` 完成数据读写。

## 数据流程

添加账单流程：

```text
AddAccount 页面
→ AccountForm 表单收集金额、分类、日期、备注
→ AccountService.insert()
→ AccountModel.insert()
→ RDB 数据库存储
→ 首页和统计页重新查询数据
```

首页查询流程：

```text
Home 页面出现
→ AccountService.getAccountByMonth()
→ AccountModel.queryByDateRange()
→ AccountService.groupByDate()
→ 页面按日期分组渲染
```

统计查询流程：

```text
Stats 页面选择月份/年份
→ AccountService.getAccountByMonth() 或 getAccountByYear()
→ 按日期或分类统计
→ 页面展示统计结果
```

## 技术特点

- 使用 ArkTS 声明式 UI 构建页面。
- 使用 Stage 模型管理应用能力和生命周期。
- 使用 RDB 实现本地持久化，不需要后端服务。
- 使用 dayjs 处理日期格式化、月度范围和年度范围。
- 使用组件化结构拆分页面，提高代码可维护性。
- 使用断点系统适配不同屏幕尺寸。
